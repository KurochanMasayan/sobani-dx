# 手動発注リクエスト

## テーブル概要
- **テーブル名**: 手動発注リクエスト
- **説明**: 手動で発注リストに追加するための入力テーブル
- **主キー**: リクエストID
- **役割**: 手動発注追加のフォーム入力用。オートメーションで商品マスタに反映される。

## カラム定義

| カラム名 | 物理名 | データ型 | 必須 | 列種別 | 説明 |
|---------|--------|----------|------|--------|------|
| リクエストID | request_id | UniqueID | ✓ | 物理 | 一意識別子（自動生成） |
| 商品ID | product_id | Ref | ✓ | 物理 | 商品マスタ参照 |
| 手動発注数量_箱 | manual_order_boxes | Number | ✓ | 物理 | 発注数量（箱単位） |
| 作成日時 | created_at | DateTime | ✓ | 物理 | レコード作成日時 |
| 作成者 | created_by | Email | ✓ | 物理 | 追加した担当者 |
| 商品名 | product_name | Text | - | 仮想 | 商品名（計算式: [商品ID].[商品名]） |
| 発注先名 | supplier_name | Text | - | 仮想 | 発注先名（計算式: [商品ID].[発注先名]） |

## AppSheet設定

- **ラベル**: [商品名]
- **キー**: リクエストID
- **リレーション**: 商品ID → 商品マスタ.商品ID
- **初期値**:
  - 作成日時 = NOW()
  - 作成者 = USEREMAIL()

## フォーム設定

手動発注リクエストの追加フォーム：
1. 商品ID（必須）- 商品マスタからRef選択
2. 手動発注数量_箱（必須）- 数量入力

## オートメーション連携

このテーブルにレコードが追加されると、オートメーションで商品マスタが更新される。
詳細は「18_手動発注オートメーション.md」を参照。
