# 患者マスタ

## 概要
訪問歯科診療の対象となる患者の情報を管理するマスタテーブル

## テーブル定義

| カラム名 | データ型 | 必須 | 説明 |
|---------|---------|------|------|
| ID | Number | ○ | 患者を一意に識別するID |
| 氏名 | Text | ○ | 患者の氏名 |
| カナ | Text | | 氏名のフリガナ |
| 生年月日 | Date | | 生年月日（YYYY/MM/DD形式） |
| 性別 | Enum | | 男/女 |
| 施設ID | Ref | | 施設マスタへの参照（FAC + 3桁連番） |
| 部屋番号 | Text | | 施設内の部屋番号 |
| 保険種別 | Enum | | 生活保護/後期高齢/国保 |
| 負担割合 | Enum | | 0割/1割/2割 |
| 重度医療 | Text | | 無/有/非課税 等 |
| 要介護度 | Enum | | 非該当/要介護1〜5 |
| 介護負担割合 | Enum | | 0割/1割 |
| 自己負担有無 | Text | | 無/有（1割）/有（2割）/有（障害/初診料のみ） |
| 訪問頻度 | Enum | | 月1/月2 |
| 曜日 | Text | | 訪問曜日（例：ケア（月・水・金）、治療（月・水）） |
| 目的 | Text | | 訪問目的（義歯洗浄、虫歯治療、口腔ケア等） |
| 事業所ID | Ref | | 居宅介護支援事業所マスタへの参照（CMP + 3桁連番） |
| 備考 | LongText | | 担当CMの連絡先等 |

## リレーション

- **施設マスタ** ← 施設ID（多対1）
- **居宅介護支援事業所マスタ** ← 事業所ID（多対1）
