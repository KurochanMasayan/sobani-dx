# 施設マスタ

## 概要
歯科診療を行う施設（グループホーム、介護施設、個人宅など）の情報を管理するマスタテーブル

## テーブル定義

| カラム名 | データ型 | 必須 | 説明 |
|---------|---------|------|------|
| 施設ID | Text | ○ | 施設を一意に識別するID（FAC + 3桁連番、例: FAC001） |
| 施設名 | Text | ○ | 施設の名称 |
| 郵便番号 | Text | | 郵便番号（XXX-XXXX形式、〒なし） |
| 住所 | Text | | 施設の住所 |
| 電話番号 | Phone | | 施設の電話番号 |
| FAX番号 | Phone | | 施設のFAX番号 |
| 管理者/担当者 | Text | | 施設の管理者または担当者名 |
| 備考 | LongText | | その他メモ |

## 特記事項

- 「個人宅」は施設に属さない患者の訪問先として使用
- 施設名に（マル①）（マル②）などの識別子が付いている施設は関連施設
- 複数の担当者がいる場合は「、」で区切って記載

## データ件数
現在13施設が登録済み
