# 居宅介護支援事業所マスタ

## 概要
患者のケアマネジメントを担当する居宅介護支援事業所の情報を管理するマスタテーブル

## テーブル定義

| カラム名 | データ型 | 必須 | 説明 |
|---------|---------|------|------|
| 事業所ID | Text | ○ | 事業所を一意に識別するID（CMP + 3桁連番、例: CMP001） |
| 事業所名 | Text | ○ | 事業所の名称 |
| 電話番号 | Phone | | 事業所の電話番号 |
| 担当者 | Text | | 担当ケアマネージャー名 |
| 備考 | LongText | | その他メモ |

## リレーション

- **患者マスタ** → 事業所ID（多対1）
  - 1つの事業所に複数の患者が所属
