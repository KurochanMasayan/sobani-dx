# スタッフマスタ

## 概要
訪問歯科診療を担当するスタッフ（歯科医師、歯科衛生士）の情報を管理するマスタテーブル

## テーブル定義

| カラム名 | データ型 | 必須 | 説明 |
|---------|---------|------|------|
| スタッフID | Text | ○ | スタッフを一意に識別するID（STF + 3桁連番、例: STF001） |
| スタッフ名 | Text | ○ | 氏名 |
| 役職 | Enum | ○ | 歯科医師 / 歯科衛生士 |
| 電話番号 | Phone | | 連絡先 |
| 稼働曜日 | Text | | 稼働可能な曜日（例：月火水木金） |
| 備考 | LongText | | その他メモ |

## Enum定義

### 役職
| 値 | 説明 | 算定ルール |
|----|------|-----------|
| 歯科医師 | 歯科医師 | 居宅療養管理指導：月2回まで |
| 歯科衛生士 | 歯科衛生士 | 居宅療養管理指導：月4回まで、20分以上滞在必須 |

## リレーション

- **診療スケジュール** → 担当医師ID（1対多）
- **診療スケジュール** → 担当衛生士ID（1対多）

## 特記事項

- 役職によって算定ルールが異なるため、スタッフ登録時に必ず役職を設定すること
- 稼働曜日は、スケジュール作成時の参考情報として使用
- 1回の訪問で医師と衛生士が同時に担当する場合もある
